# Eunus Bot Debian
Questo repository è sincronizzato da un bot di OpenClaw denominato **GithubEunus**.
Contiene tutte le informazioni inerenti le configurazioni, le migliorie e ciò che ho imparato durante la configurazione di OpenClaw.

# Cos'è OpenClaw
[OpenClaw](https://openclaw.ai/) è un sistema agentico che si interfaccia a modelli diversi di intelligenza artificiale e che impara nel tempo in base alle interazioni del proprietario.

E' capace di creare e gestire subagenti, interagire con il sistema operativo, creare workspace etc.
