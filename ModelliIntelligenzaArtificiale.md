# Modelli di Intelligenza artificiale
Il fulcro di un sistema agentico è senza dubbio il modello di intelligenza artificiale che si sceglie.

Ciascun modello è collegato al provider tramite una chiave API che sono a pagamento.
Esistono diversi modelli:
- Anthropic
- Google Gemini
- DeepSeek
- Kimi
